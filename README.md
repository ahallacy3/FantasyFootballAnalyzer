## Project Prereqs ##
* Python 3.7, pip, virtualenv (pip and virtualenv should be included with a python install)
* Python in your PATH

## Project Setup ##
* clone
* virtualenv venv
* activate venv
  * On Windows, venv\Scripts\activate
  * On Mac/Linux, source venv/bin/activate
* pip install -r requirements.txt
